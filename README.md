### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 视频分享
5. 视频排名
6. 交流论坛
7. 留言反馈
8. 个人中心
9. 我的发布
10. 我的收藏
11. 后台管理

**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 用户管理
4. 视频分享管理
5. 视频排名管理
6. 平台公告管理
7. 视频类型管理
8. 交流论坛
9. 留言板管理
10. 系统管理（轮播图管理）
